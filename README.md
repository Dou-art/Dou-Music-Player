           1  + # Dou 网易云播放器 🎵
           2  +
           3  + 一款基于 NeteaseCloudMusicApi 的现代化网易云音乐播放器，拥有精美的 UI
              设计和流畅的动画效果。
           4  +
           5  + ## ✨ 功能特点
           6  +
           7  + - 🎨 **精美界面** - 流光背景、毛玻璃效果、动态光效
           8  + - 🎵 **完整播放** - 播放、暂停、上一首、下一首、进度控制
           9  + - 📝 **歌词显示** - 支持逐字歌词(YRC)和普通歌词(LRC)，带动画效果
          10  + - 🔍 **搜索功能** - 搜索歌曲、歌手、专辑
          11  + - 📋 **歌单浏览** - 推荐歌单、排行榜
          12  + - 🖥️ **全屏模式** - 沉浸式播放体验
          13  + - 🔄 **播放模式** - 顺序播放、随机播放、单曲循环
          14  + - 💾 **播放列表** - 查看和管理当前播放队列
